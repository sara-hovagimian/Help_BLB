Planilla para Ganancias

Esta tarea permite diseñar modelos de informes que incluyan los datos exigidos por la AFIP de modo que puedan presentarse como Declaración jurada de impuesto a las ganancias.

Para ingresar estas novedades el usuario no debe tener tildado Denegar para la acción Ver datos de la liquidación para los legajos y las variables que se desean informar. Para más detalle sobre este tema se debe consultar el apartado Como definir los permisos .

General

 Año impositivo

Informado un año impositivo, la planilla devuelve la última liquidación en el año que se indique.

Para los legajos que no tienen ninguna liquidación en el año que se indica no se incluye información.

 Rectificativa

Activando el campo se emite una planilla rectificativa. Esto significa que se incluirán las novedades de ganancias informadas para planillas rectificativas o de tipo ambas.

 Legajo

Códigos correspondientes a los legajos para los que se emite el informe.

En caso de indicar más de uno, se puede utilizar el - para indicar rangos y la , para separarlos.

Si se deja en blanco se incluye a todos los legajos.

 Atributo

Código del atributo en base al cual se emitirá el informe.

Número

Número que identifica al concepto.

 Valor

Valor del atributo correspondiente a los legajos para los cuales se emitirá el informe.

 Grupos

En este cuadro se ingresan los grupos definidos en la tarea correspondiente que permiten diseñar el informe.

 Código

Código del grupo que se incluye en el informe.

 Descripción

Exhibe la descripción del grupo informado.

 Tipo de reporte

Se debe seleccionar el tipo de planilla que se desea emitir.

 Planilla de ganancias

 Formulario 649

Transf. Elec. AFIP

Desde esta ficha es posible realizar la exportación de los del Formulario 1357. El archivo destino deberá ser luego importado en la AFIP.

 Presentación AFIP

Habilita la exportación de datos para la transferencia electrónica de AFIP.

 Ubicación archivo

Establece la ubicación destino donde se guardará el archivo generado.

 Solo remuneraciones mayores a:

Permite exportar sólo la información de los Legajos que superen la remuneración informada.

 Tipo de Presentación

Permite definir el tipo de presentación que se desea exportar.

 Emisión

 Para tener en cuenta:

Los grupos se ordenan numéricamente. Por lo cual, si se debe incorporar un nuevo grupo a un informe ya existente, debe asignarse al mismo un código tal que responda a la ubicación deseada.

Al emitir la planilla, en el cuadro del encabezado :

 

Como fecha inicial del rango se imprime la fecha de ingreso (si ingresó en el año que se lista); en caso contrario sale el día primero del mes. Como fecha final del rango se imprime la fecha de egreso (si el legajo egresó en el año que se lista) y, en caso contrario, el último día del mes.
